# Microservices Architecture Playground

A production-oriented collection of microservices design patterns implemented using Spring Boot. Each module demonstrates a real-world architectural pattern used to build scalable, resilient, and maintainable distributed systems.

## Tech Stack

- Java
- Spring Boot
- Apache Kafka
- PostgreSQL
- Docker
- REST APIs
- Maven

## Implemented Patterns

- JWT Authentication
- Secure Service-to-Service Authentication
- Kafka Event-Driven Communication
- Saga Pattern (Choreography)
- Saga Pattern (Orchestration)
- Outbox Pattern
- Event Sourcing
- CQRS
- Aggregator Pattern
- Sidecar Pattern

## Learning Order

If you're studying these patterns from scratch, I recommend following this order:

1. JWT Authentication
2. Kafka Event-Driven Communication
3. Saga Pattern (Choreography)
4. Saga Pattern (Orchestration)
5. Sidecar Pattern
6. Outbox Pattern
7. Event Sourcing
8. CQRS
9. Aggregator Pattern
10. Secure Service-to-Service Authentication
